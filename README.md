# Junk or No

A beginner-friendly Next.js app that helps users decide whether a typed food item is junk food or not.

## Features

- Built with Next.js, JavaScript, and TailwindCSS.
- Simple food input for beginners.
- Friendly result cards for junk food, not junk food, and unknown foods.
- Example buttons so new users can test the app quickly.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
